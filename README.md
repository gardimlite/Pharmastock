About
Pharmastock adalah aplikasi request restock obat yang memudahkan staff apotek dalam hal logistik obat, aplikasi ini juga adalah project UAS matakuliah pemrograman mobile.

Installation
1. Download folder UASmobile
2. extract
3. masukan 'pharmastock_admin' dan 'pharmastock_api' kedalam XAMPP/Laragon atau database yang anda pakai
4. buat tabel database bernama 'db_pharmastock' di phpmyadmin
5. import 'db_pharmastock.sql' kedalam tabel database anda
6. buka 'http://localhost/pharmastock_admin/'
7. install 'app-release.apk' di hp android anda
8. buka aplikasi yang sudah diinstall
9. pilih daftar akun dan buat akun baru
10. beralih pada 'pharmastock_admin' di localhost anda, lalu klik kelola user, lalu approve request permintaan registrasi baru
11. login ke Pharmastock dengan akun yang sudah anda buat
